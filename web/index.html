HOW TO PLAY

SHERLOCK HOLMES

JAMES MORRIARTY

Tugas anda adalah melarikan diri dari Sherlock Holmes. Anda tidak boleh menggambil object terakhir

GENERAL

Dalam game ini terdapat 2 mode, Harus atau Tidak boleh menggambil object terakhir, untuk memaikan game ini anda hanya memilih object menggunakan mouse dengan syarat boleh mengambil berapapun object dalam satu baris
